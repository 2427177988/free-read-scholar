`free-read-scholar` 是一个免费的学术论文搜索和阅读平台。它旨在整合全球学术数据库资源，让用户能够便捷地搜索和访问学术论文，特别是开放获取的全文。项目包含前端（React）和后端（Node.js），后端代理外部学术API（如NCBI）以获取数据。其核心目标是打破学术壁垒，促进知识自由流通。

## 技术栈

*   **前端:** React
*   **后端:** Node.js (Express)
*   **代理服务器:** `my-ncbi-proxy-server` (Node.js)

## 功能

*   学术论文搜索
*   论文详情展示
*   访问开放获取的全文链接
*   ...

## 安装与运行

1.  **安装 Node.js 依赖:**
    ```bash
    # 进入后端目录
    cd my-ncbi-proxy-server
    npm install
    # 进入前端目录 (假设前端在项目根目录或另一个子目录)
    # cd frontend-directory
    # npm install
    ```

2.  **启动后端服务:**
    ```bash
    # 在 my-ncbi-proxy-server 目录下
    npm start
    ```

3.  **启动前端开发服务器 (如果适用):**
    ```bash
    # 在前端目录下
    npm start
    ```
